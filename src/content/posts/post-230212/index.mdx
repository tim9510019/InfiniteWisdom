# 導言:
你是否很心急，想要趕快獲得成功，但是往往事與願違。漸卦告訴我們，沒有任何事情是一蹴可及的，都會有一整個過程。讀懂漸卦有助於我們理解按部就班的重要性，時機未到不用心急，繼續用正面正向的態度，堅定往目標前進。
按照次序不求急，當然也不會故意去慢，是一種漸進之道。古時候，婦女要出嫁，要有所歸宿，等待男方主動提出來，才能答應，透過婚姻舉例來告訴我們，不可求快，要循序漸進，如果按照漸卦去走，嫁女兒沒有不求吉利的。女人以外為歸，需要一步一步發展，不要急，家庭要先互相了解一下，兩家的背景有沒有需要調和的地方，男方工作也是要循序漸進，步步高升，之後才能撐起整個家。

# 彖曰:
## 漸之進也，女歸吉也。進得位，往有功也。進以正，可以正邦也。其位剛，得中也。止而巽，動不窮也。
進去了，有位置了，才能有功勞，才會慢慢步步高升。要求工作機會要光明正大，不要走後門，進去要好好做，不要混，有往前發展的力量，跟精神，進以正，目標正確，不偏不倚，到最後這種精神，可以用來治國，可以正邦也。
其位剛，得中也。止而巽，動不窮也。宰相必起於州部。做任何事情，把目標搞正確了，方向搞明白了，態度很良好，順順的走，一步一步地往前推進。做事自然會順。

# 象曰:
## 山上有木，漸。君子以居賢德善俗。
循序漸進，一層一層的往上，一步一步地堆出來的。君子以居賢德，善俗。用權勢或用法律途徑都不好，要靠教化，移風易俗，易經告訴我們要發揮參考權，讓大家互相學習，周遭的人互動以後會有所感應。一旦偏差誤解扭曲以後就會開始亂了，如果沒有按部就班，就要好好地調整過來。

上善若水，水的進展，都是由近到遠慢慢推出去，循序漸進，現象給我們做參考用的，要體會現象背後的道理，也就是本體本質，也就是道理。從艮卦來，但是很像否卦，否卦之所以否，是因為第三爻第四爻不溝通，藉由34爻交換，漸漸溝通，最後可以達到泰卦的境界。漸卦提醒我們雖然已經意識到要一步一步來，但是由於很多因素導致一出手就錯了，很快地調整過來，由時候雖然每一步都很謹慎，可是也有可能會得意忘形，會搞得亂七八糟，六個爻用鴻雁來比喻，有一定的次序，知道何時要遷徙，要耐得住，碰到缺陷的時候，自己要知命。

作為一個女性，要找到合適的對象，作為一個男人，要有辦法負起整個家庭的生活責任。雙方按照共同的理想，循序漸進。如果只是互相利用一段時間，那就不是鴻雁了，做任何事情要有固定的目標。自己的正道就在那邊，現在走歪路，是方法調整，是為了讓走正道的路更順，要對自己目標的忠貞不二。

# 初六:
## 鴻漸于干，小子厲，有言，旡咎。
## 小子之厲，義旡咎也。
鴻雁往來有時，先後有序，做人做事要有條理有次序。鴻漸於干(水岸相接處)，小子厲。有言，無咎。出入仕途，剛剛找到工作，處於初六爻的不當位，上面沒有人支持你，也沒有人幫助你，倒處都會有人有傳言，要"正"，講了很多不利於自己的話，該做的就做，不該做的就不做，小子之厲，義(合乎禮儀)無咎也。老人會想辦法把新人擠走，自保，所以新人會不當位，雖然有危險，但是要想辦法站穩。
# 六二:
## 鴻漸于磐，飲食衎衎，吉。
## 飲食衎衎，不素飽也。
受排擠受壓力受冤枉，還站得住腳。鴻漸於磐，飲食衎衎(和樂的狀態)，吉。經過努力有很好的表現，進一步的得到入位。飲食衎衎，不素飽也(不要尸位素餐)，要好好工作，工作完了以後，對於職位沒有不安的感覺，持續往前好好的努力，有表現，是吉順的。適可而止，控制自己，避免讓老人吃味。
# 九三:
## 鴻漸于陸，夫征不復，婦孕不育，凶。利御寇。
## 夫征不復，離群醜也。婦孕不育，失其道也。利用御寇，順相保也。
鴻漸於陸，夫征不復，婦孕不育，兇; 利御寇。鴻雁飛過來在水邊了解環境，在磐石上也站穩了，直到上了陸地，離群了以後，就會越來越危險。夫只知道出征，不知道回家，導致婦生而不育。知進不知退，誰也控制不了他。夫征不復，離群醜也，婦孕不育，失其道也，利用御寇，順相保也。一個人在團體裏面，要守團體的秩序，一旦離群奮鬥，如果會用人的人，會把你外派，或是讓你去完成一個小目標，避免和別人處不好。
# 六四:
## 鴻漸于木，或得其桷(ㄐㄧㄝˊ)，旡咎。
## 或得其桷，順以巽也。
或得其木角(ㄐㄧㄝˊ)，順以巽也。當一個人被老闆外派的時候，被老闆安排總是做小案，要反省，是否自己太好高騖遠，只按照自己的想法去做，不顧整體的合作。這時候如果調整過來，還能夠無咎。
# 九五:
## 鴻漸于陵，婦三歲不孕，終莫之勝，吉。
## 終莫之勝，吉。得所願也。
鴻漸於陵(山上高峰)，婦三歲不孕，終奠之勝，吉。鴻鳥是向南飛，飛過雁門關，飛行的道路裡面最高峰的，只要飛過雁門關以後，就回到家了。就算夫婦一段時間有了誤會，隔閡，但是只要雙方都是忠貞的都會化解，還是吉順。在一個機構機關裡面，從基層往上爬有順有不順，只要你永遠走正道，大家會知道，該給你的會給你。
# 上九:
## 鴻漸于逵，其羽可用為儀，吉。
## 其羽可用為儀，吉。不可亂也。
鴻漸於陸，其羽可用為儀，吉。登峰造極以後，慢慢要往下，要回到原來的地方，人一定會老，事情也是有極限的，隨時都要有功成身退的準備，要端端正正的實實在在地保持風範，告老還鄉以後，維持以前的那種風範，做任何事情，不緩不急按部就班，貪戀祿位，不知道功成身退，都是凶的。上台不管怎麼樣，下台下的很漂亮，就可以當典範，接下來就進入到下一個卦歸妹卦，待下回分曉。
