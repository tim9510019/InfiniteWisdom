# 導言:
常常聽到謙虛是美德，但是過度謙遜，謙卑忍讓又常常會喪失做人的原則，甚至被人欺負，到底該如何做，才能進退得宜，不失個人風骨又不致於與人硬碰硬？這些都是巽卦可以教會我們的人生處世之道，就讓我們開始探索吧！
巽卦，上下兩卦都是風在吹，形容要完成一個政令宣導，必須先帶起風向，什麼風呢？底層民心所向，對於渴求改變的氣氛。
上層對於底層人民的需求有放在心中，融合上層所掌握的多方意見與資訊，經持續調整之後，再藉由不斷的宣導與提醒，才有辦法達到全國人民都能夠遵守並改變行為的結果。
巽卦不是告訴我們要事事謙卑，處處忍讓。大自然中可以看到，風吹草伏，風吹草，草會順，風不吹草的時候，就會原形畢露，直挺挺地站立著，做人或許就要像那草一樣。當形勢比人強的時候，不要硬碰硬，要先順服，但從來沒有忘記自己的原則，找時機可以慢慢的移風易俗，以身作則，逐漸地改變組織與團體的風氣。
巽卦的錯卦是震卦，揭示如果父母對於子女溝通，講得很輕柔，兒女未必會順你，要震，要驚嚇，警醒了以後才會開始順從。玉琳哥的熱炒店裡面就有講到，提醒兒女，門會夾到手，沒有任何效果，但只要拿小黃瓜示範被門夾到會斷掉，兒女馬上受到驚嚇，就能記起教訓。

# 雜卦傳:
## 兌見而巽伏也。
巽卦的縱卦是兌卦，兌卦表示將喜悅之情，展現於外，人人都看得到，相對於巽卦，則是需要把情緒先隱藏在心中，不要及時地展現出來。由於時機不對，情況不對，場合不對，不宜當下與人硬碰硬，可以等到冷靜的時候再慢慢溝通調整。
要切記，第一時間要管制自己的脾氣，避免說錯話，做錯事，與人起衝突，沈默是金，冷靜下來再做任何回應。處事原則先放在心中，事後慢慢的去改變一切。
巽卦不是告訴我們要無條件地順從，任何事情都是向反面發展，當一件事情變壞了，還要去順從他，也沒有任何道理可言，做人也沒有盡到責任。當人事物是符合客觀自然規律，正當的，可以謙順。當不正當的時候，也可以不必當面硬碰硬，用順的方式，而有原則，慢慢影響人，移風易俗。
事事順人是沒有人格的人，事事頂撞的人是沒有修養的人。
# 彖曰:
## 重巽以申命，剛巽乎中正而志行，柔皆順乎剛，是以小亨，利有攸往，利見大人。
重巽以申命，要不斷的宣導，別人才會聽得懂，記得住，才做得出來。剛巽乎中正而志行，剛指的就是95，很中正很合理，有這個意志，就可以全力來推行。柔皆順乎剛，柔要順剛的道理，就是因為順著才會改變人，如果不順就無法深入，於是會變得拿別人一點辦法也沒有。
有合理的目標，有堅強的意志，要實施理想必須不折不饒，所以只能小亨通，無法通順。以柔來順剛，其中還是要有一些方法，才有辦法應對進退，並不是傻傻的無止盡的順從。
無論再怎樣的謙順，也只能小亨，當你對別人很謙順，別人就不把你當一回事，開始得寸進尺，最後也不會大亨。用這種態度做事，雖會很有利，但是要找到有修養的大人物，才能發揮很好的效果，而不會只是被當消耗品，不斷的被利用。
# 象曰:
## 隨風，巽，君子以申命行事。
告訴我們政府的政令要推行之前，要不斷的宣導，人民才會聽得懂，記得住，才能實行得出來，做任何政策，不可以過於急促，疏於宣導，將導致天下大亂。
風是連續相隨的，一陣一陣的，下卦對現行的不滿意提出建議，要移風易俗，要改變現況，所以上卦，要做適當的調整，率先做出典範，推行出去，上下配合，就很容易移風易俗。君子以申命行事，反覆喻曉，一再告知，要實施一段時間，讓別人養成習慣，才可以說已經徹底執行了一個政策。
同時，政府與人民的風向，每一個政府機關都要把握住。要針對同一個政令採取同一個步調，口徑一致，老百姓才會比較容易接受，這也是巽卦上下吹風的原因，吹的是同樣的風向，這樣才不會讓所有老百姓與公務員神經錯亂。
# 序卦傳:
## 旅而無所容，故受之予巽，巽者入也。
行旅在外搞到沒有容身之地，表示自己沒有融入他鄉的環境，也吃盡了苦頭。這時候要謙遜待人，入境隨俗，與人為善。
陽光普照大地，但是只是照到表面而已，稍微有一點阻檔就無法深入。不像風，因其柔順而能無孔不入。旅居他鄉，如果處事不夠圓滑，很容易不被當地文化所接受，這時候要放下身段與人為善，漸漸的才能發揮自己的影響力，被當地的人逐漸的認同。
巽卦陰爻在下，代表底下的聽話臣服，父母叫得動小孩，老師可以分派一些作業，統治者的政令很容易施行，令出必行。
巽卦告訴我們，要看看父母給自己的命令是否合理，老師給的作業份量是否適當，政府的政令是否替老百姓著想，如果合理就要順從。如果不合理，就要有一套方法來不順，這就是巽卦的宗旨。
當合乎天理的命令出來，順從會成為君子，當違背天理的命令出來，順從就會成了小人，巽卦不是告訴我們要以卑下立身，而要依天道而行。
老子曾說，柔弱勝剛強，巽卦，四個陽爻擋不住兩個陰爻。要柔順，要有方法，而非受到刺激，就不柔了。巽卦擺在旅卦之後，就是告訴人們，你吃盡苦頭了，你還敢嘴硬，你還敢感情用事，最後是倒霉的。
要記取教訓，用理智來指導感情，巽不是永恆不變的立身之道，而是要我們擇善而從，內方外圓，隨時都能夠保留靈活調整的空間。
巽卦的卦主，物以稀為貴，上下陰爻都是卦主，以柔為主，要謙順，同時也是居卑下以行事的指導原則。
巽卦是一種圓通，而不是告訴你要長期卑下，是短期的靈機應變，而不是長期的做人原則。巽卦的中心思想，就是謙順有為，這個才是真正的巽卦宗旨，如何謙順還要有為，就有賴對於每個爻的理解。
巽這個卦，上下卦都是陰卦主，但是位置都是很低下的。表面上看起來是由上而下，雷厲風行，實際上是順應民意，順應需要，所以要從下卦第一爻看起。絕對不是上卦，在位者單方面覺得要如何做就如何做，完全沒有考量到下層的狀況就能夠成事。
# 初六:
## 進退，利武人之貞。
## 進退，志疑也；利武人之貞，志治也。
陰爻不當位，地位很卑微，或許才能也不高，進退兩難。年輕人提出改革，沒有人理會，畏首畏尾，也不太敢決斷，其實是因為身份地位，環境因素，使得他進退兩難。利武人之貞，如果卑下之人要反映問題，不可以進退不定，只要自己的意志力不夠堅定，就無法成事。
志治，治理自己的意志，一再重複自己的意見。改變與一切移風易俗的工作，需要年輕人的意志。年輕人意志要堅定，把自己的定位做好，道理原則守好，才能獲得別人的重視，要隨時想辦法振奮自己的士氣，表現出勇敢意志，不可以有任何膽怯的模樣。
# 九二:
## 巽在床下，用史巫紛若，吉，无咎。
## 紛若之吉，得中也。
九二是整個改變的主導者，有謀有勇，推動改革，巽在床下，上床的人份量大，下床的人份量不大，用史巫，紛若。要有多種意見融合，要虔誠的向神明禱告，必須祭天才能服人，要表示做任何決定，都自己不是一個人說得算的。
宣示有向自然學習，也尊重自然規律，然後如果真的都做到位了，就會很吉順。紛若之吉，得中也，中是宇宙的自然現象，也就是合理，綜合大家的意見，找到有道理的共識，就些就是九二該做的事情，有點像是現代 KOL (民間意見領袖）該承擔的責任，如此就能逐漸推動整個社會的向上提升。
# 九三:
## 頻巽，吝。
## 頻巽之吝，志窮也。
頻巽之吝，志窮也，英雄剛強是剛強，但是沒有這個權勢，勢不夠強，志不夠堅定，然後採取的手段過於激烈，大家對你感到反感，甚至適應不良，達不到任何效果。這告訴我們，移風易俗的工作不可操之過急，揠苗助長，反倒會栽跟斗。
# 六四:
## 悔亡，田獲三品。
## 田獲三品，有功也。
六四陰爻在陽爻之上，柔承剛，被兩個陽爻夾擊。表示，無論對上對下要發揮柔順。悔亡，所有的後悔都消失了，知道該怎麼應對進退了，就沒有任何後悔了。由於做好承上啟下的工作，田獲三品，有功也，有很大的收穫，必定馬到成功。如此一來可以獲得很多的獵品，可以面面俱到，便可以做到對上面有交代，對下面可以打交道，就沒有任何悔了，對於整個事情的推動還功不可沒。
# 九五:
## 貞吉悔亡，无不利，无初有終，先庚三日，後庚三日，吉。
## 九五之吉，位正中也。
貞吉，悔亡，無不利。先聽聽下卦的暈染，看看整個過程，先旁觀，心裡非常清楚，該如何調節，先想想要如何順應民心，滿足各方面的需求，這樣一下就貞吉了，就悔亡了，就可以無往不利。無初有終，還是有些人有意見，不可能每個人的意見都照顧到，會讓人家感覺自己偏哪邊，會有意見。但只要做到先庚三日，後庚三日，也就是事前宣導，事後緩衝執行，如此一來，成果都會很好。下命令要變更之前，會努力去溝通協調。正式執行以後，還會給大家留有執法的緩衝地帶。無論做什麼事情，都要嚴中有寬，寬中有嚴。這告訴九五，對此該為則為，不必有過多的顧慮，只要做法有條有理，符合天道，都是大吉大利。
# 上九:
## 巽在床下，喪其資斧，貞凶。
## 巽在床下，上窮也；喪其資斧，正乎凶也。
巽在床下，喪其資斧，貞兇。做一個領導人，最重要的就是要有決斷力，或著是權力，當斷則斷，不然的話就亂了。上九表示，如果連這種決斷的工具都丟掉了，卑順過了頭，即使人很正，最後後果也不好。
即使一個人很正，但是卻很卑順，這樣就沒有人會把你當一回事，這樣沒有任何決斷力，也沒有人理你，就不會有任何作為，別人也是欺負你。
當時局混亂，想改變一切的有志之士，知道自己不能過度順從，但是要剛要強，又會立刻得罪人，到底該怎麼辦，才能兼顧兩者呢？
有原則要堅持，但是表情要和顏悅色，要輕鬆一點，不要把別人一下子就嚇到了，所以接下來要討論兌卦，與人和悅相處之道。
