# 導言:
為政者必須要有誠信，這樣政令要百姓有所節制的時候，百姓才會接受，反過來講，只要自己克制節制，就必定能夠對人誠信，這是一體兩面。於是，節卦之後，易經就安排了中孚卦，告訴人們，無論立身，還是處事，都應該心懷誠信，而且要廣施以仁，不可以居心不誠，虛情假意。這幾乎是人人都知道的道理，那為什麼不容易實踐? 中孚卦中的孚，其實是俘虜的時候，當一個人被俘虜，深受險境的時候，才有辦法看出這個人是否是誠信的，太平時期，所有人都說得很好聽，歷史上的例子數不勝數，但是，也無法隨意批評人，是否誠信，只有運用，謙卦，要謙虛地，去了解各種情況，綜合考量以後，才有辦法做出判斷。

# 卦辭:
## 中孚，豚魚吉，利涉大川，利貞。
合理的誠信，並不是老老實實，誠誠懇懇，過於老實，反而非誠信且接近愚昧。豚魚吉，代表小豬小魚，非常難以馴服，如同小孩也是過於好動，控制不了自己，但是只要自己有所節制，即使是最冥頑不靈的生靈，也都能因此而有所感應，化解大難，因此能，利涉大川。但是，面對極度遲鈍的人，也不能說，為何對其推心置腹，居然如此被對待，然後就生氣，反目成仇，這就不是中孚了。利貞，即是告訴我們，中孚，恆常的保持不偏不倚的態度，有利於持續堅持守正。如同，孔子總是不言其煩的，回答各式各樣的問題，然後針對不同的受眾，給予不同程度與背景的回答方式，這就是中孚接近完美的示範。

# 雜卦:
## 中孚，信也。節而信之，故受之以中孚。有其信者必行之。
要相信還是不相信，沒有對錯的問題，僅僅是上過當以後，要警惕的問題，人和人之間要講誠信，恰恰是因為，彼此都不信，人類沒有道德以後，才說仁義重要，沒有仁義以後，才說禮智重要，仁義禮智都沒有了，只能講誠信，可見，現代人最可憐。中孚，就是合理的相信，合理的誠信，如同獨木舟，不可盡信，有可能會翻船，所以要節制地相信，即是，我相信不相信你是我的事，你誠懇不誠懇是你的事，獨木舟和坐上他的人，沒有辦法事互相完全保證，只能說雙方有合理的信任，合理的誠信，合理的保證，這樣就能夠一起向前滑行。誠心不妄，只要持續不產生妄念，時常能夠知道節制，保持中庸恆常，這樣就永遠能夠一直中孚下去。

# 彖曰:
## 中孚，柔在內而剛得中，說而巽，孚，乃化邦也。豚魚吉，信及豚魚也；利涉大川，乘木舟虛也；中孚以利貞，乃應乎天也。
中孚卦內外是鏡射，誠心即是心，內外一致，如同鏡面反射，柔在內指的是陰爻在六三六四，而九二與九五皆陽爻，得中，整個就是陽包陰的卦，內心要柔，但外表要有一致性，不可搖擺不定，要剛。講誠信，必須要剛柔並濟，不能盡信，也不能全不信。下卦是兌卦，代表喜悅，上卦是巽卦，代表風，代表要順從，能夠喜悅的順從，才代表中孚的力道，是夠強大的。中孚誠信能輕而易舉的感化整個邦國，因為內柔外剛，能夠讓人在很喜悅的同時，遵守奉行所有約定事項，而非只是一團和氣，嘻嘻哈哈，目無章法。豚魚吉代表好的領導，要連那些難以感化的豚魚，都能受到你的感化，誠信要能滲透團隊裡面每一個分子。乘木舟虛也，代表整個中孚卦，中間是陰的是空的，如同獨木舟，木頭的中心是空的，這樣才有辦法載人，才有辦法利涉大川。同時，中孚的原則就是中庸恆常，要長久的保持，而非變來變去，翻來覆去，一個獨木舟，之所以會有人敢坐上去，即是知道，獨木舟不會翻來覆去。能夠長久恆常的，必然是能夠順應天道的法則，即是中孚能利貞之原因。如同大自然隨時在變，但是有規律，春夏秋冬，日出日落，月出月落，東昇西降，每天都是太陽與月亮。人但凡要變，要求新求變，都必然要有一些核心原則不變，就和老天依樣。如果連春夏秋冬，連各種基本規律都可以改變，那世界上，就沒有甚麼是可信的。孔子論語說，天何言哉，四時行焉，百物生焉，老天爺從不說話，但是所有人都對老天爺是有信心的，因為四季從來沒有亂過，對百物是很公正的，如好雨知時節，隨風潛入夜，潤物細無聲，普遍的給予支持與協助。

# 大象:
## 澤上有風，中孚，君子以議獄緩死。
澤上有風，風來感動水，水就隨著他去產生波動，這裡面完全沒有人為的矯揉造作，純粹是自然的，無偏見，很公正，即是中孚。君子看到這個景象，便會領悟出一個道理，君子以議獄緩死，不可以廢除死刑，但是可以，去無限延長執行死刑的時間，避免有人冤死，無法挽回。同時，也不能放過那些窮凶惡極，這樣對被害者家屬，完全不公，沒有誠信。中孚卦正中間是陰柔是虛的，代表誠信，在於柔軟的內心中，不在於嘴巴上，所以要多加討論，開誠布公，內心要謙虛，要中需，廣納大家的意見，沒有利害關係，才能叫做誠信。

# 初九:
## 虞吉，有它不燕。
## 初九虞吉，志未變也。
虞即是，要很仔細的去考察，去思慮，合乎大道的去相信，就要專心去信，就要不疑有他，深信不疑，就去實踐，不燕即不像燕子依樣，到處反覆地飛來飛去。既然答應人家，即使吃虧上當，也要志向不變的做下去。即使，吃虧了，也只是告訴自己下次要警惕，不要一開始就隨便相信人家，避免變成是自作自受而已。處信的開始，就要好好的明辨，避免變成愚忠，要仔細辨別有沒有價值，有沒有符合宇宙大道，有道理，然後無論有錢沒錢，都會選擇性，明辨後才相信，這才叫君子。

# 九二:
## 鳴鶴在陰，其子和之；我有好爵，吾與爾靡之。
## 其子和之，中心願也。
九二，陽居陰位，常常喊破喉嚨，看似也沒有人呼應，但是持續呼籲以後，必然會有同頻共振的人，會相互呼應，所以根本無需慌張，只要是恆常的大道，長久下來，沒有不被買單的道理。即使是孔子，也僅僅是鳴鶴在陰，這麼好的建議，在春秋戰國時代，也是無人搭理，孔子尚且如此，平常人又何必著急? 越是誠信堅守天道，越是有人會把他的東西傳承下去，上天自然會莫名其妙的，在歷史上封給你一個位置，那就是被上天封的爵位，即是天爵。照道理去走，可能在正業上虧錢，但是老天會在別的地方，莫名其妙的補償你。道德經即說，這即是天地無親，常與善人，即是老天，吾與爾靡之。歷史上，奉獻最大的往往都是九二，老天給我，即是有之，老天不給我，本來就沒有這個想法。

# 六三:
## 得敵，或鼓或罷，或泣或歌。
## 或鼓或罷，位不當也。
六三不當位，被六四所壓制，六四就會壓制六三，六三陰居陽位，與上九相互呼應，總是被六四擋住去路，僅僅是因為同性相斥，共同爭取95與96的認可，不當位的六三，居於下風，要不作戰，要不作罷，狼狽而逃，要不悲傷哭泣(輸慘)，要不歡樂而歌(險勝)，得失行之於色，處中孚的環境，但是不誠信，必定是不斷反覆，如同偽君子那樣，比起偽君子，真小人相對令人喜歡，至少其立場不會持續反覆。各種人都有，外小人內君子，或室外君子內小人，還有小人變君子，君子變小人，隨時都在變動。身處社會之中，往往只能自求多福，自己的腳要站得穩，誠心不妄，才有辦法免去各種牽來扯去的麻煩，因為，誰對誰錯，根本是沒有人說得明白，講得清楚的。

# 六四:
## 月幾望，馬匹亡，无咎。
## 馬匹亡，絕類上也。
月亮快要滿，但是沒有盈，明明有很好的馬給你騎，但是馬匹死了，六四和九五很親近，六四只要莫名其妙地開始對初九好，也就是人民好，九五就會懷疑是否六四要取而代之，好好的馬匹，其實是被自己所害死。六四以為自己有資源，要開始爭取民心，老闆都會懷疑他的忠心，但只要想辦法切得乾乾淨淨的，好雨知時節，為善不欲人知，都能夠無咎。君王很相信64派他去賑災，當一片百姓反應都很好的時候，君王反而會擔心，這六四倒底要幹嘛，這即是功高震主。歷史上做得最好的是賈詡，敬告家人，不能和任何人有來往，也不能讓任何人，讚許自己，這即是身處六四爻的智慧。

# 九五:
## 有孚攣如，无咎。
## 有孚攣如，位正當也。
九五剛實居中，剛直陳信，緊握完全不肯放鬆，無私，即為天下者，不顧家。要自己節制，才能夠去節制別人，九五與九二不相應，九二鳴鶴在陰，九五不欣賞，但是仍然要保持一樣的態度，使其發聲，但仍然未必要買單，始終如一，貫徹始終。當一個地區，所有人都不講話了，那肯定要越來越亂了。君王如果要買千里馬，又怕沒有人惜售，便差遣九二去買，九二卻買回了一匹死掉的千里馬，這也是鳴鶴在陰，所有人都看不懂，這到底是在做甚麼，但是隨即，所有人都看到，死的千里馬都買了，那活的你更要買了，留著九二不殺頭，反而能起到奇效。

# 上九:
## 翰音登于天，貞凶。
## 翰音登于天，何可長也。
當鳥不斷往上飛，會很淒慘，這樣很累的時候，也沒有地方可以棲息，就只能哀鳴了，九六不當位，沒有能力又要充能幹，就像那個飛鳥一樣，孤立無援，虛名無法持久，翰音登于天，隨風高飛，虛而不實。大名之下，難以久居，所有員工，都是要測試老闆，講真話到底會不會生氣，但是不可太過，一定要懂得寬留餘地，這種智慧，就在小過卦裡面，另行分解，敬請期待。
