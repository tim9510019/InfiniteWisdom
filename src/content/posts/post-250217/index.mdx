# 導言:
小過即是稍微有一點點過分，稍微有一點點過失，理論上都是不好的，但如果連一點點小過失都不敢犯，人生也過於拘謹，將會導致甚麼都不敢嘗試，搞得自己寸步難行。理想上，人人皆希望出拳能夠命中目標，但現實總是談何容易，在沒有犯任何小過失的情況，也就無法從中學習要如何拿捏那個度。同樣的，過於年輕與過於老邁，皆是無法拿捏好這個尺度，有太多種種不理想因素，如棒球選手與籃球選手就有很明顯的年齡障礙，投籃與投球表現會受影響。由於，有太多諸如此類不可控因素，導致人人都面臨各種過與不及，種種痛苦的經驗。
小過卦即是告訴我們，在某些狀況之下，即使稍微有點過失，有點過分，那都是好的，善惡也不是如此分明與固定的。大部分情況，父母都要讓小孩自由表現，適切的嚴厲責備小孩，寧可在家裡面被父母罵，也不要出去外面給別人惹笑話。小過卦最大的意義，是鼓勵我們勇敢的去做事情，不要怕犯錯，如果不敢做，不願意做，最後必然是不能做。為了怕錯而不敢做，為了怕丟臉就甚麼都不敢做，那就是故步自封了，如果整個社會都這樣的話，那社會怎麼會進步?
只要萬變不離其宗，始終傳遞同樣的道理，那這樣都稍微嘗試一下，絕對是沒有關係的，這便是，小過不斷，大過不犯，平常都藉由不斷的積極嘗試，不斷地犯小錯誤，來取得經驗的積累，便能逐漸遠離風險，最終才會有辦法恰到好處，不犯大的過錯。
當主管的，有些事情必須寬厚一些，如果完全不許犯錯，那下屬極有可能就乾脆不做，不做就絕對不會犯錯，那這樣也是注定無法達成組織的目標，實際該如何拿捏尺度，要如何順勢而為，就必須針對每一個爻來做分析，才有辦法見分曉。

# 卦辭:
## 小過，亨，利貞。可小事，不可大事。飛鳥遺之音，不宜上，宜下。大吉。
小過有利於堅持自己的原則，在小事情寧可過分一點，但是在大事情，千萬不能過分。損失小，傷害輕，擔當的起，那就是小過，比如講真話被打爛考績，這種小事，反而能少繳很多所得稅，但如果是自己完全承受不起的傷害，那也根本無需去做嘗試。小過卦的卦辭，很巧妙地用飛鳥來做比喻，小過卦中間兩個陽爻很像鳥的身體，上下皆是成雙的陰爻，很像鳥的兩雙正在飛的翅膀，於是有飛鳥遺之音的比喻，任何飛鳥，都不能一直任性飛得過高，或是飛到脫離群體，若如此，就必然哀鳴，哀鳴就叫做遺之音，根本不知道誰在哀鳴，只聽到聲音。要犯小過，不宜上，宜下，不能逆著世界潮流與大勢去做，順著方向做，若是過頭，還拉得回來。但若是逆勢往上衝，絕對拉不回來。不宜上，僅僅是說不要逆勢去操作，而非警惕不可觸犯上位，同理，宜下，僅僅是說順勢把事情做好，而非順勢去欺負弱小，只要把握好時機，小過又能承受，必然是大吉的結果。

# 雜卦 / 序卦:
## 小過，過也
## 有其信者必行之，故受之以小過。
雜卦的部分，難道是說，就算是很小的過，也是過錯? 並非如此，小過在某些情況算是過失，在某些場景，僅僅是經過，是過程而已。序卦的部分則是說，如果是經過心存誠信，問心無愧，同時也很審慎地加以選擇是否要誠信，一旦果決推行下去，就難免矯枉過正，產生過失，故受之以小過。某些場景，過分一點可以，但是不能把過分當正常，矯枉過正，終究不是普通年代的常態。

# 彖曰:
## 小過，小者過而亨也，過以利貞，與時行也。柔得中，是以小事吉也。剛失位而不中，是以不可大事也。有飛鳥之象焉，飛鳥遺之音，不宜上，宜下。大吉。上逆而下順也。
小的事情稍微有一點過分，但是仍然能夠亨通，即使過分了，也還能夠利於堅持守正，為何能如此，必然是配合時機，順著大勢再走。歐美吹起精簡人事的風氣，美國DOGE更是重用年輕人，進行官僚體系的清掃，如果遇到自己老闆，過於保守，用官威或是各種手段，壓制與控制年輕人，便能順勢而為的給予建議與指教，最差最差就是當年的考績比較差，但是事過境遷以後，仍然是會亨通。道理很簡單，如果都沒有任何一個人給出意見，單位也可能大概率，會被人事精簡掉，長久來看，小過仍然會是亨通的。柔得中，彼此呼應，心心相應，都認為這是小事，很柔地去有點過分，而非是大手大腳進行各種實質上的傷害，這種小過失必然會吉祥。同時要警惕，又因整個大環境對自己是不利的，所以做事心態，能做就做，九三與九四，因剛失位而不中，註定做不了大事，無須過於掛心要成就多大的事業。不用過於想要往上高飛，避免飛得極高，哀鳴無人知，找準時機就要順勢往下飛，上方注定是逆的環境，往下飛即是吉祥。

# 大象:
## 山上有雷，小過。君子以行過乎恭，喪過乎哀，用過乎儉。
雷打在高處，例如山上，普通百姓皆生活在平地居多，沒有造成多大的過錯，造成的傷害沒這麼大。君子看到這種狀況，要謙恭的時候，寧可過度一點。處喪禮，絕對要哀傷，不能笑。寧可平常稍微減約一點，都不要過分。待客可以寬一點，但是對自己可以節約一點。這樣，窮的時候，才會有朋友來看你，平常寧可對自己刻薄些，對朋友好些，不可反其道而行之。小過卦，上下都是兩個陰爻，中間是兩個陽爻，陰的部分有點過度。上下兩個卦，震卦象徵過度是動的，艮卦象徵停止，即適可而止，在過與不及之間，應該做好合理的拿捏。

# 初六:
## 飛鳥以凶。
## 飛鳥以凶，不可如何也。
陽位陰爻，陰居陽位，才能不夠又要冒充很能幹，和九四相應，才會造成很凶的結果。初六，剛出社會，才能不夠，但是去攀關係，攀親戚，有關係，請託一下，就得到這樣得機會，明明能力不夠就承擔重任，一直往上飛，知道打鐵要趁熱，但是能力不足，只能在高空中哀鳴。這種過錯注定是不小的過失，黃金時間一過，甚麼都無法彌補，也甚麼都無法補救，道歉也沒有用。

# 六二:
## 過其祖，遇其妣。不及其君，遇其臣，无咎。
## 不及其君，臣不可過也。
六二很守本分，當位又居中，把九三當成自己的父親，九四當成祖父，六五當成祖母，要把自己當成是君的臣，一家上市公司裡面最大的便是董事長了，只能說底下的長官有所過失，但無論如何，都不可往公司最大的君位去指摘，一切過錯都要這些長官與下屬主動去承擔起來，承擔久了，反而沒有過失。爭功諉過一說其實是相反的，過錯與任何不合理對待，下屬要主動承擔，最後必然是回到長官身上，因為長官是最後最後負責人，有功勞，其實要推給別人，最終，反而自己也有一份，絕對不會有人真的冤枉你。

# 九三:
## 弗過，防之，從或戕之，凶。
## 從或戕之，凶如何也。
九三有止過的責任，應該要防止小過，而非去和上六相應，遵從上意，為非作歹，上面的人永遠是有權無責，一旦做錯，沒有人會替你擔任何責任，這種情況註定會極度凶險，痛苦難堪。

# 九四:
## 无咎，弗過，遇之，往厲必戒，勿用永貞。
## 弗過遇之，位不當也；往厲必戒，終不可長也。
陽居陰位，動卦下爻，顯示無論是能力或是德行，都足以來防止小過，所以一開始就說無咎，不會越位去行事，弗過，即言行不會過分，遇之，即一切合乎本分，不可越位，一旦越位必然要有所警惕，永遠要記住，那怕位高權重，和六五相比還是人微言輕，由於不當位，用力往前衝，注定也是不可長久堅持的，即是，除非有特殊的情況，才能去做，才能去闖看看。

# 六五:
## 密雲不雨，自我西郊，公弋取彼在穴。
## 密雲不雨，已上也。
六五是君位是柔的，陰居陽位，不當位，故不可大事，很悶很不開朗，烏雲密布，蒼生盼著要有雨，但是雨始終下不來。六五從郊外來，因為不當位，要不斷反省自己，不要把責任都往下推卸，底下幹部都很強，偶爾要替下面多想一想。公弋取彼在穴，明明在君位，但是不稱為君，稱為公，雖然是大權在握，但是能力不足，很多事情在六五看起來是好的其實是壞的，看起來是壞的其實是好的，於是六五做任何事情都注定要掂量掂量。六五在小過卦裡面，過了兩個陽剛，又不當位，那更是難當，下屬都比自己強，這就是小過卦，要宜下不宜上，要不斷地向下請教，而非努力的飛鳥遺之音。

# 上六:
## 弗遇，過之，飛鳥離之，凶，是謂災眚。
## 弗遇過之，已亢也。
遭遇到高飛不下，已經過於超過，發出哀鳴，事態極度嚴重，所有飛鳥都已遠離，極度凶險，天災人禍。當領導的人高亢，尚且有救，但若非是龍，如上六是陰的，為人臣，極度越權，不該管的，管得過於多，與初六相比，初六是失職，上六是越權，到底哪個可怕? 越權遠比失職可怕，失職請人來補一補即可，越高階的人越怕越權，位高權重且為人臣者，極度擴張權力慾望，極度凶險。
看完六個爻以後，若非順勢而為，功高震主犯上絕對是凶險的，然後被長官錯怪，反而是吉祥的，被誤會若能永遠中孚，萬變始終不離其宗，吾道一以貫之，最終定是沒有甚麼不能化解。大多數人皆認為創業是艱難的，但其實守成更不容易，隨時有小過錯出現，隨時也需要不斷與時俱進，這樣才能把豐碩的成果守住，這即是既濟卦的內容，敬請期待，下回分曉。
