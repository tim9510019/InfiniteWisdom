# 導言:
人生是場戲，也是一趟旅途，不管路途上風景如何，終有結束的時候。該如何好好地走完這趟旅途，演完這場戲，有賴易經旅卦，帶給我們一些啟示。
人人都是來地球做旅客，人人都不是地球的主人。從旅卦之中可以知道人生之道，知道如何安身立命。上一回我們討論了，丰卦，當親戚朋友都來了，很多很久不見的朋友也來了，表示你發達了，不過很快地就可能會迎來反轉。於是，窮大者必失其居，人人看了眼紅，都想要把你拉下來，故易經序卦傳說，窮大者必失其居，故授之以旅。丰卦走到最後，有可能連家都顧不了，家破人亡是常有的事情，但絕對不可以喪志，人生的旅途仍要繼續走下去。
卦辭，旅，小亨，旅貞吉，處在旅卦，心態只能求小亨，無法求大安與久安。如果一個人被迫離開他的國家，人生地不熟，無法大亨，只能小亨。
# 彖曰:
## 旅，小亨，柔得中乎外而順乎剛，止而麗乎明，是以小亨，旅貞吉也，旅之時義大矣哉。
彖辭，柔得中乎外，而順乎剛。講的是65居中，柔爻處在上下剛爻之間，旅行在外要跟當地人處得好，不要過分要求，看不順眼，可以小亨。處在顛沛流離的狀態，做什麼事情都要適可而止，下卦艮，就是告訴人要止，因為離鄉背井，出外不比在家。止而麗乎明，要請教當地的事情，一定要找對人，不然鐵定會上當。旅卦初六跟九四都不當位，上九根六五也不當位，將近2/3都不當位，處境非常得可憐。旅，貞吉也，如果你守正，如果你按照這兩個原則，還能夠吉祥。深一層會去看，旅行會有很多危難，並不一定是享受，開開心心的，所以出門在外要特別小心。旅之時義之大矣哉，在旅行的時候可以學到很多東西，可以鍛鍊你的才智，有許多磨鍊的機會，這樣才叫做行萬里路，讀萬卷書，搞懂旅行的意義，可以讓一個人的精神提升，意義重大。
但是要小心，旅卦也會毀滅一個人的心智，本來很豐富的資源，現在會發現種種都是缺乏的，如果本來處於高高在上的地位，現在因為國破人亡，跌落下來，心情會有很大的不適應，如果只求苟安，失去了志氣，就會喪志了。
# 象曰:
## 山上有火，旅，君子以明慎用刑而不留獄。
大象，山上有火，旅，山上的火常常因為旅行而意外導致，例如在生火的時候，不小心搞到火燒山，火是很明的，小火可以照明可以取暖，當火很猛力的時候會被燒死，君子以明慎用刑，要有威嚴才能領導，但如果過分的苛刻，人都會跑光了。而不留獄，領導者還是要明辨是分，要審慎考察去執法。不要一直沒有做下判斷，讓老百姓的心都懸在那裡，無法持續統治。
當人由高峰跌到低谷時，心情要如何調適呢？旅是丰的不良後果，但不要喪志，勤儉，努力堅持繼續打拼，終能落葉歸根，榮登故里。
要如何旅行？很多人帶麻將出去打，帶手機出去盯手機，那何必旅行？很多導遊在外面亂講，那為什麼會亂講，因為認真講沒有人要聽，不了解自己也不了解什麼是呂，旅行是讓我們知道天外有天，每個地方有每個地方不同的生態環境，風俗習慣，要互相尊重。老子提出自賓，要把自己當成是客人，不要在外面肆意妄為，要徵求當地人的同意，要移風易俗，融入進去。吃當地的菜，穿當地的衣服，要用欣賞的心態。如果到一個比我們發達的地方，我們是賺到，要由衷感謝，但是也不能回來羞辱國人，也不要外國的月亮比較圓。
旅居在外的原則都分布在旅卦的爻辭裡面，不管是要度假，逃難遠離戰爭，要旅居海外，都要遵守這些原則來行事，就能夠小亨。失人失時失位的卦，要格外的謹慎小心，才有辦法度過危險。
在異鄉的過程中，稍有不慎就會遭致災禍，要如何一步一步度過難關？
# 初六:
## 旅瑣瑣，斯其所取災。
## 旅瑣瑣，志窮災也。
陰爻可以看成是被動，是跟隨別人去旅行，沒有辦法決定行程。旅瑣瑣，思齊所以災。初六不當位，只憑一個人要出去闖天下，是做不到的，在旅途當中有很多瑣碎的事物要自己料理，千萬不要因為這些難題而懷憂喪志，旅瑣瑣，志窮災也。告訴我們不要因此而失去志氣，反而會給自己帶來災難，必須要想辦法改變，想說這只是暫時的。初六雖然和九四相應，但是沒有人要伸出援手。由於九四陰承剛，也是自顧不暇，然後初六自身也不成材。
# 六二:
## 旅即次，懷其資，得童僕貞。
## 得童僕貞，終无尤也。
六二，旅既次，在旅途的當中，得到居住的居所，不管是旅館還是人家家裡的客廳也好，懷其資，得童僕貞。可以擺脫旅瑣瑣的糾纏。
# 九三:
## 旅焚其次，喪其童僕，貞厲。
## 旅焚其次，亦以傷矣。以旅與下，其義喪也。
九三，旅毀其次，喪其童僕，貞厲（危險）。華僑去到別的國家，剛開始是受盡委曲，慢慢的想要揚眉吐氣，要參與當地的政治，想要主導一切，政治是當地人的事情，基本上只能求在當地跟別人處得很好，做做生意，很守份，不要不滿足於現實，搞到最後房子也被燒掉了，跟你的人都跑光了。旅毀其次，亦以傷矣，以旅與下，其義喪也。搞到一切的東西都丟光了，這是因為你過分了。要站的位置高一點，想得遠一點，適可而止。到當地作客，不要讓當地人有壓力，你要來搶地盤，爭取民心，當地的君王或是統治者，絕對是做不安穩，其義喪也，已經失去了做旅人該要守的本分。孔子絕糧於陳的故事告訴我們，一個人有才能的旅人，如果對當地人稍有不利，就有可能落到無地安身，食不果腹的境地。
# 九四:
## 旅于處，得其資斧，我心不快。
## 旅于處，未得位也；得其資斧，心未快也。
九四，象徵有才能的人，被君王流放出去，日子不好過。這種人知道，這些地方都是他暫時居住的地方。旅於處，得其資斧，我心不快。只有拿到工具，但都不是可以使用的資源，還得自己動手去做，做可以勉強出頭，不做死的很快。
# 六五:
## 射雉，一矢亡，終以譽命。
## 終以譽命，上逮也。
六五，射雉一矢亡，終以譽命，旅行終究不是長久的事，要順利地離開，終結我們的旅途，一個人最後要心安理得，要得好死，把一生好好的了斷。終以譽命，上（下）逮也。柔爻，要小心翼翼地親近左鄰右舍，到任何地方，一定要知道主人是誰，這是別人的地盤。
# 上九:
## 鳥焚其巢，旅人先笑後號咷，喪牛于易，凶。
## 以旅在上，其義焚也；喪牛于易，終莫之聞也。
上九，鳥毀其巢，一個旅客如果過分高亢自大，就好比巢下面有火，旅人先笑後嚎啕，喪牛于易，凶。牛是最容易馴服的一種動物，但是你都丟光光，當然會兇，到別人的地盤不要耀虎揚威，高高在上，危險性非常的高。在別人的地盤，屋簷下，不可以高亢。猶太人住五星級飯店，但是吃得很隨便，住要安全，但是吃的還好。要用入，用順的態度來過一生，所以旅卦的後面是巽卦，要柔得其宜，要順應一生的遭遇，知道什麼是合理什麼是不合理的，這樣才能好好的過一生，下回分解巽卦，敬請期待。
