# 導言:
既濟和未濟卦彼此的關係，相對於其他卦，相對複雜許多，既是錯卦，還是綜卦，也是交卦，根據易經的解讀原則，簡單的要複雜化，複雜的反而要簡單化。既濟卦和未濟卦都是由坎卦和離卦組成，只是位置彼此交換一下。既濟卦是內心光明，但是外在環境很多坎險，很多因素會導致失敗，然而，未濟卦則是，遭遇坎險，即使外在情勢一片大好，也沒有多大用處，處於泥菩薩過江，自身難保的狀態。但是，未濟卦不是不濟，要看成是尚未成功，身處險境，反而會格外警慎，充滿著希望，與其早成功不如晚成功，早成功會與人爭名鬥勝，反而會陷自己於不利的境地。綜觀易經64卦，一切都是階段性調整的故事，吾十有五兒志於學，要學會謙卦，謙虛的向老師學習，同時也要留餘地給老師，老師才願意教導你，三十而立，就是離卦，一個人最起碼要把大自然的現象，大致上有所了解，看到光明面，能夠立定為人處世的目標，四十而不惑，第40卦就是解卦，問題都解了，自然不惑。五十而知天命，第50卦就是鼎卦，知道有任何小小的成就，必然要虔誠的謝天，不然往往是無攸利。六十耳耳順，第60卦就是節卦，知道要節制自己的慾望，不該聽的不該看的，會耗費心神的，都知道要有所節制。七十從心所欲，不踰矩，把六十四卦的易理，放在心中，不斷隨緣來回味，然後隨時拿來回味發生過的現象，培養覺察，與看戲的心態，最終必然能領會，宇宙無限智慧，無論如何都可以，都不會超越該有的尺度。然後，對於占卜，往往僅僅是一步之差，就能從成功到失敗，或是從失敗到成功，未濟與既濟僅僅是一個爻的輪動而已，人生就是不斷的水深火熱，在水裡的時候希望伙來烤乾衣服，不過當上岸以後，反而會懷念在水裡的涼快，於是就如此不斷水深火熱，不斷循環往復，至死方休。不過最終的最終，無論是誰，從哪裡來，最終必然歸於虛無，然後再次生生不息。於是，了然於心，無論遇到何種境地，必然能永懷希望，這也是能從易經最後一卦，未濟中的六爻，學習到的智慧。

# 卦辭:
## 未濟，亨。小狐汔濟，濡其尾，无攸利。
未濟，潛藏著無限發展的生機，相比於既濟只有小亨，或著亨小，來的長遠的多，遭遇坎險，或是掉到坑裡，僅僅是鍛鍊自我的意志，事情尚未完成，或是組織面臨調整與重組，才有年輕人或是有志之士，努力發揮的空間。處在未濟的環境下，如同小狐狸想要渡河，但是無奈尾巴也沾溼了，很重，不斷被扯後腿，然後一無所成，這告訴我們，隨時要量力而為，天時地利人和要具備，才有自己能夠發揮發展的空間，然後要比一般人，更堅持，更小心，更忍耐，不能夠隨便讓尾巴濕了，要保護好自己。


# 雜卦 / 序卦:
## 未濟，濟也
## 物不可窮也。故受之以未濟，終焉。
既濟以後，許多招數都使出來了，江郎才盡，然後無論有多少財產，總有一天會花光，總有一天會體會到，從甚麼都沒有的地方，到甚麼都沒有的地方，未濟卦會變成易經的第六十四卦，就是告訴我們，一切都是循環往復，才能夠生生不息。前人的辛苦耕耘，前浪死在沙灘上，僅僅是給後人提供更好的條件，登高望遠而已，於是有了，未濟，濟也，雜卦傳的說法，一切都僅僅是過渡中，任何故事其實都能夠，接近無止無盡，不斷延續下去。


# 彖曰:
## 未濟亨，柔得中也。小狐汔濟，未出中也。濡其尾，无攸利; 不續終也，雖不當位，剛柔應也。
未濟只要堅持到底，注定是亨通的，因為坎險，柔得中，一切都僅僅是磨練，即使是小小的狐狸，也知道，渡河看起來很難成功，但是仍然沒有放棄希望，即使是無攸利，沒有悠哉就能得利的機會，或是根本沒有機會，但是只要不間斷的堅持，永不放棄(不續終)，終有機會去打破現有狀態，絕對不要懷憂喪志。未濟卦，沒有一個爻是當位的，但其實無須傷心，剛與柔皆相應，遇到問題就是檢討，然後改進，不要自暴自棄，盡人事以後，一切都僅僅是隨波逐流，無攸利即是能不能成功，都是老天的事情。就算沒有結果，完整經歷整個過程，都是極佳的人生閱歷。


# 大象:
## 火在水上，未濟，君子以慎辨物居方。
火是向上的，水是向下的，如果位置擺錯，根本沒有交集，火也完全沒有辦法起到，烹調的作用。物要以類聚，人要以群分，要很慎重地去辨別，萬事萬物是否與目標同道，要辨明清楚，並且居方，擺在正確的處所。同道可以擺進一點，不同道可以敬而遠之，一切都是沒有定下來，不是不能完成，而是尚未完成而已。把所有資源，各種所遇到的問題，謹慎地辨別分析處理，妥善配置，放在對的地方。


# 初六:
## 濡其尾，吝。
## 濡其尾，亦不知極也。
初六是位置最低下的，居於坎卦的最下方，心裡十分不安，但是初六與九四相應，於是想找九四幫扶，無奈九四不當位也不居中，容易導致自不量力，然後全身從頭到尾都沉了，不是只有尾巴進到水，於是會導致吝，與亦不知極也，太不知道自己的極限，太不知道自己的能力。


# 九二:
## 曳其輪，貞吉。
## 九二貞吉，中以行正也。
未濟卦，九二以陽居陰位，和既濟卦的踩煞車，意義有所不同，未濟卦的踩煞車，是避免過於不自量力，消耗過多體力，居中但是不當位，往往會過於陽剛，需要忍耐，再看看其他狀況，再做打算。九二是陽爻，但是六四是陰爻，這會導致一種弱主強臣的處境，容易胡作非為，只能中正以行，不受邪魔歪道引誘，才會貞吉。雖然在坎坷的環境當中，但是始終堅持合理行為，不為任何誘惑所引誘，那必然會貞吉。


# 六三:
## 未濟，征凶，利涉大川。
## 未濟征凶，位不當也。
即使找到脫險的機會，也是不能大意，雖然六三與上九相應，但是中間仍然有一陰一陽阻擋，不是那麼容易就能爬過去的，於是為征凶。未濟的環境裡，走到六三的位置，已經精疲力竭，即使好不容易有脫險的機會，也是很容易被淹沒，於是要學習，九二的精神，穩紮穩打，慢慢的利涉大川，反而能一路順暢。所以身處六三，不要心急，即使有人拋繩子過來，也要多多考慮再先，畢竟相應的上九，不居中也不當位。


# 九四:
## 貞吉，悔亡。震用伐鬼方，三年有賞于大國。
## 貞吉悔亡，志行也。
九四已經開始脫險，於是一開始就告訴你貞吉，如果堅持走正道，堅持不變的原則，就會很順利，九四本來是不當位的，但是由於守規矩，貞吉，所以無悔。震用伐鬼方，則是告訴我們不要和其他人一樣同流合汙，要保持好自己，把能力表現在壤外，而不是在內鬥，持續堅持下去，就會逐漸光明，就會得到很大的獎賞。但是畢竟，九四是剛強不當位，六五是個弱主，極度容易功高震主，就會立刻完蛋，於是這裡告訴我們要將功高震主的這股力量，拿去征服外夷。六五主弱，一旦功高震主，就是六五眼中的鬼方。處於這種境地，不表現被懷疑，一表現很多人嫉妒。之所以能夠，貞吉悔亡，即是運用了志行也的智慧，行得正，意志堅定，不受誘惑改變初心，沒有私心，就算大家對我一時有誤會，最後仍然會吉祥。


# 六五:
## 貞吉，无悔。君子之光，有孚，吉。
## 君子之光，其暉吉也。
六五以柔行剛，沒有辦法表現自己的能力，很多的股東都想要欺負你，於是要發揮君子之光，也就是謙讓謙虛，有孚眾望，大家心裡有你，才會順遂才會無悔。冬天的太陽是溫暖的，但是夏天的太陽是酷暑，變成霸王即是不尊重別人，不當位，陰居陽位，就要藉力使力，需要大家的協助，而不是自己去發揮甚麼光輝。


# 上九:
## 有孚于飲酒，无咎。濡其首，有孚失是。
## 飲酒濡首，亦不知節也。
經歷所有坎險，最終迎來光明，所有人都很快樂，此時不飲酒不唱歌，更待何時，但是不能過度，不可忘記過去那種奮鬥艱難的過程，千萬要提醒自己，人類是健忘的動物，錯誤導致危難，其實會不斷發生，從未濟走向既濟，也要記得未濟整個過程中，所得到的教訓。適時的慶功宴，當然是無咎，但是不能過於沉浸其中，避免濡其首，整個泡在酒缸裡面。很多人，一但事業成功，就天天去喝酒，不知節制，以為一切都無須再操心，然後養成壞習慣，就會不知不覺又掉回坎險裡面。人生注定是坎離坎離，不斷水深火熱，這是上天的美德，告訴我們步步為營，安步當車，盡人事，聽天命。盡人事，期許自己能夠多複習易經，把自身修好，然後聽天命，如果運氣夠好的話，能夠帶動家人，朋友與整個社會，朝向更和諧中道的方向去前進。
